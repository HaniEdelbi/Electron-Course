# Dark Mode Toggle

This folder contains the dark mode theme for the toggle button.

## Features
- Dark color palette with deep blues and warm oranges
- Stronger shadows for depth
- Optimized for dark backgrounds
- Sun icon glows with golden tones
- Moon icon has white glow effect

## Usage
Import this CSS file when the app is in dark mode:
```html
<link rel="stylesheet" href="../design/dark-mode/toggle.css">
```

## Color Scheme
- **OFF state**: Deep navy gradient (#1a1d2e → #16213e)
- **ON state**: Warm orange gradient (#ffa500 → #ff8c00)
- **Thumb**: White with grey gradient and subtle shadows
