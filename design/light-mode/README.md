# Light Mode Toggle

This folder contains the light mode theme for the toggle button.

## Features
- Lighter color palette with soft blues and yellows
- Brighter shadows and glows
- Optimized for light backgrounds
- Sun icon glows with warm yellow tones
- Moon icon has subtle blue tinting

## Usage
Import this CSS file when the app is in light mode:
```html
<link rel="stylesheet" href="../design/light-mode/toggle.css">
```

## Color Scheme
- **OFF state**: Soft indigo/blue gradient (#e8eaf6 → #c5cae9)
- **ON state**: Bright yellow gradient (#fff9c4 → #ffeb3b)
- **Thumb**: Pure white with subtle grey gradient
